# Announcement: aospdtgenUI

I'm excited to announce **aospdtgenUI**, a fork of the original aospdtgen tool with several new features designed to make Android device tree generation easier and more powerful!

## Key Features:
- **Graphical User Interface (GUI):** A user-friendly Tkinter-based interface for those who prefer not to use the command line.
- **Dual ROM/Recovery Support:** Generate device trees that are compatible with both Custom ROMs and Custom Recoveries (like TWRP) simultaneously using the `--dual-support` flag.
- **OTA Update Support:** Easily specify an OTA URL for your LineageOS-based builds.
- **Built-in Update System:** Check for and install tool updates directly from within the GUI.

## Installation:
```sh
pip install aospdtgenUI
```

## How to use:
Simply run `aospdtgenUI` to launch the graphical interface, or use the CLI for automated tasks.
